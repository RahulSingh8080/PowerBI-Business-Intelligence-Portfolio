This project presents an interactive **Hospitality Analytics Dashboard** built in Power BI, designed to help hotel stakeholders monitor performance, identify trends, and make data-driven decisions across revenue, occupancy, booking behavior, and room performance.

---

## Project Overview
This dashboard analyzes key hospitality metrics and provides actionable insights related to:
- Revenue performance
- Occupancy % trends
- ADR (Average Daily Rate) and RevPAR (Revenue Per Available Rooms)
- Realisation % and booking patterns
- Room category contributions
- Property-level comparisons
- Customer demand patterns

The goal is to deliver a **single source of truth** for hotel managers and decision-makers.

---

## Key Objectives
- Analyze revenue distribution across categories
- Track weekly and monthly demand patterns
- Compare performance across properties
- Understand booking platform efficiency
- Provide leadership with actionable insights

---

## Key KPIs Included
The dashboard highlights the following essential KPIs for **Atliq Grands**:
- Total Revenue
- RevPAR (Revenue Per Available Room)
- DSRN (Daily Sold Room Nights)
- Occupancy %
- ADR (Average Daily Rate)
- Realisation %

These KPIs measure operational efficiency, demand, and revenue management performance.
